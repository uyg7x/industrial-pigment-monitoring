# Pigment Process Monitoring Suite

A professional Python desktop application for pigment manufacturing monitoring.

## Features
- Process monitoring dashboard
- COD and BOD prediction
- Pigment color prediction
- Equipment failure risk analysis
- Batch management
- Inventory tracking
- Delivery tracking
- Alerts and recommendations
- Charts and Excel export

## Run
```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
python run.py
```

## Recommended Python
Python 3.10.x
